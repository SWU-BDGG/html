# html

BDGG 프로젝트 웹 프론트

## 필요한 페이지

1. 로그인 페이지
    - 아이디
    - 비밀번호

2. 회원가입 페이지
    - 아이디
    - 비밀번호
    - 비밀번호 확인

3. 파일 업로드 페이지
    - 파일 업로드

4. 파일 검색 페이지
    - 검색 결과를 보여줄 목록

5. 파일 목록 페이지
    - 업로드된 파일을 보여줄 목록
    - 이전 페이지로 넘어가는 버튼
    - 다음 페이지로 넘어가는 버튼
    - 페이지 숫자 보여주는 거

6. 오류 페이지
    - 403 오류 : 로그인이 필요함
    - 404 오류 : 페이지를 찾지 못함
    - 기타 오류
        - 업로드할 파일을 전달 받지 못함
        - 업로드할 파일의 크기가 너무 큼
        - 검색 결과가 없음
        - 다운로드 할 파일을 찾지 못함
